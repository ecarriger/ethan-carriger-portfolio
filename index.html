About Me

Ethan Carriger is a frontend web developer that specializes in designing and developing websites and web applications using modern web programming languages and frameworks. From a simple WordPress site for small business entrepreneurs to powerful custom React web applications, Ethan has the depth and range of experience and development skills to provide you with excellent service for your specific web needs.

Ethan's skill at problem solving and love of learning new things make him especially suited to overcoming obstacles and finding solutions to whatever challenge is in front of him. An example of this is when a client wanted a mobile friendly companion app for their curriculum, so Ethan created a hybrid web app that he was able to package and publish on the Google Play and Apple App stores for the client's user to have as an app on their phones or tablets.

Beyond his web development profession Ethan enjoys playing board games with friends, playing guitar on his church's worship team, and making lattes for his wife and himself.

Jammming

React

HTML

CSS

UI/UX

Jest

Jammming is a web app that connect to the Spotify API and allows users to look up songs and create playlists for Spotify. The project was prompted by my Codecademy course to demonstrate my ability to design and develop fully featured web apps using React. I started by designing the UI in Figma, then developed the frontend using React and published using GitHub Pages. I implimented automated testing using Jest and React Testing Library. Finally, I developed a backend to proxy the API requests which I hosted on AWS.
